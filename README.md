## 📚 Descripción del Proyecto
  
El objetivo principal es crear un **gestor de notas académicas** que permita al usuario registrar, consultar y eliminar calificaciones de diferentes cursos.

## ⚙️ Funcionalidades
- Agregar estudiantes y sus notas.  
- Calcular promedio de calificaciones.  
- Modificar o eliminar registros.  
- Mostrar el listado general de estudiantes y sus notas.
